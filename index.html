PolyTrucker

PolyTrucker is an arcade low-poly truck driving game. The game features missions in which you must pick up cargo using a truck, and then deliver them, making sure to avoid any terrain and animals as you make your way to the drop off point. You are scored based on how fast you delivered the cargo, minus points for hitting animals. This game was built as a group project in CS576. In terms of production, I was solely responsible for the UI of the game, including but not limited to the main menu, pause menu and the in-game HUD. I was also responsible for creating the animal-based AI for the game, which acts as an enemy that wanders around and runs away from the truck's horn.
